Editing with ProseMirror

It takes all of two lines code to set up a ProseMirror editor:

import {ProseMirror} from "prosemirror/dist/edit"
let pm = new ProseMirror({place: someElement})

You might want to include some more modules, for example to add a menu. Menu styles are just extension modules in ProseMirror. This page includes the classical toolbar style, and a tooltip based one. You can use the control below the editor to switch between them.

Once you have an editor instance, you can call methods like getContent("html") or getContent("markdown") to get its content in a given format. Or you can directly access its internal document format and work with that.

Using ProseMirror

This is editable text. Focus it and start typing.

To apply styling, you can select a piece of text and manipulate its styling from the menu. You get emphasis, strong text, links, code font, and images.

Block-level structure is manipulated, depending on the current menu style, either through the top menubar, or through the “hamburger” menu to the right of the selected paragraph. This allows you to change paragraphs into headers or code blocks, to wrap them in lists or blockquotes, to unwrap them, and to insert things like horizontal rules.

Try using the "indent” item in the menu to wrap this paragraph in a numbered list.
